Quick Start & Build

Prerequisites

RequirementMinimumNotes
C++ CompilerC++17GCC 7+, Clang 5+, MSVC 2017+
CMake3.14
Qt5 or 6Core + Concurrent modules
OpenCL1.2Optional — required only for --device gpu

Clone & Build

# Clone with all submodules (CNN → ANN → OpenCLWrapper)
git clone --recursive https://github.com/naschpitz/NN-CLI.git
cd NN-CLI

# Build
mkdir build && cd build
cmake ..
make -j$(nproc)

# Run tests
./test_nncli

# The NN-CLI executable is ready
./NN-CLI --help
Submodule chain: NN-CLI includes CNN under extern/CNN, which in turn includes ANN under extern/ANN, which includes OpenCLWrapper under extern/openCLWrapper. The --recursive flag clones all of them.

Verify Installation

# Check that it runs
./build/NN-CLI --help

# Expected output:
# Usage: NN-CLI [options]
# Neural Network Command Line Interface
# 
# Options:
#   -h, --help          Displays help
#   -c, --config <file> Path to JSON configuration file
#   -m, --mode <mode>   Mode: 'train', 'predict', 'test'
#   ...

First Training Run

Here's the fastest way to verify everything works — train a small ANN:

# 1. Create a minimal config file
cat > tiny_config.json << 'EOF'
{
  "layersConfig": [
    { "numNeurons": 2, "actvFunc": "relu" },
    { "numNeurons": 4, "actvFunc": "relu" },
    { "numNeurons": 1, "actvFunc": "sigmoid" }
  ],
  "trainingConfig": {
    "numEpochs": 100,
    "learningRate": 0.1
  }
}
EOF

# 2. Create sample data (XOR problem)
cat > xor_samples.json << 'EOF'
{
  "samples": [
    { "input": [0,0], "output": [0] },
    { "input": [0,1], "output": [1] },
    { "input": [1,0], "output": [1] },
    { "input": [1,1], "output": [0] }
  ]
}
EOF

# 3. Train!
./build/NN-CLI -c tiny_config.json -m train -s xor_samples.json -v

Project Structure

├── main.cpp                 # Entry point, CLI argument parsing
├── NN-CLI_Runner.hpp/cpp    # Core execution logic
├── NN-CLI_Loader.hpp/cpp    # Config loading, network detection
├── NN-CLI_Utils.hpp/cpp     # IDX file format loading
├── NN-CLI_ProgressBar.hpp/cpp # Training progress display
├── extern/
│   └── CNN/                 # CNN submodule (includes ANN)
├── examples/
│   ├── MNIST/               # ANN MNIST example config
│   └── MNIST_CNN/           # CNN MNIST example config
└── tests/                   # Integration tests
